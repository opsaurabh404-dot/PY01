# Student Management System (Python)

This is a simple Student Management System project built in Python. 
It allows you to add students, view their marks, and update marks using a menu-based system.

## Features
- Add students with their marks
- View all students and their marks
- Update marks of existing students
- Simple input validation and error handling

## Concepts Covered
- Python basics (variables, dictionaries, functions)
- Loops and conditionals
- Input/output handling
- Debugging and error handling
- Simple project structure for maintainable code

## How to Run
1. Clone the repository
2. Open `student_management_system.py` in Python 3
3. Run the file and follow the menu instructions
